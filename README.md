
  INSTRUÇÕES DE USO - CONVERSOR DE IMAGENS PARA WEBP

ESTRUTURA DAS PASTAS:
- imagens_originais/  <-- Cole AQUI todas as imagens (.jpg, .png, etc.)
- imagens_convertidas/ <-- Suas novas imagens .webp vão aparecer AQUI
- converter.py        <-- O script de automação

PASSO A PASSO PARA RODAR:

1. Instale a biblioteca necessária no terminal/prompt de comando:
   pip install Pillow

2. Coloque todas as imagens que deseja converter na pasta "imagens_originais".

3. Execute o script no terminal:
   python converter.py

Pronto! Todas as imagens serão convertidas em lote para WebP na pasta "imagens_convertidas".
